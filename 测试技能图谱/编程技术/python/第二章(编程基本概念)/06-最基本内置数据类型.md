# 06-最基本内置数据类型


每个对象都有类型，Python中最基本的内置数据类型：

- 整数
  - 整数，2345，10，50
- 浮点型
  - 小数，3.14或者科学计数法 314e-2
- 布尔型
  - 表示真假，仅包含：True、False
- 字符串型
  - 由字符组成的序列。"abc"、"测试"、


**数字**

Python支持整数和浮点数，我们可以对数字做如下运算。

| 运算符 | 说明      | 示例 | 结果 |
| :---- | :-------- | :--- | :-- |
| +     | 加法      | 3+2  | 5    |
| -     | 减法      | 30-5 | 25  |
| *     | 乘法      | 3*6  | 18  |
| /     | 浮点数除法 | 8/2  | 4.0 |
| //    | 整数除法   | 7/2  | 3   |
| %     | 取余      | 7%4  | 3   |
|    **   |  幂运算         |  2**3    |    8 |


除数为0，会产生异常：
```
>>> 10/0
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ZeroDivisionError: division by zero

```

**使用divmod函数可以同时得到商和余数**

```
>>> divmod(10,4)
(2, 2)
>>> divmod(9,3)
(3, 0)
```
