# 02-双分支选择结构


语法格式如下：

if 条件表达式：
    语句/语句块1
else：
    语句2/语句块2


案例：
```
# 输入一个数字，小于10，则打印该数字;大于10，则打印“数字太大”　

num = input("请输入一个数字：")

if int(num)<10:
    print(num)
else:
    print("数字太大")
```


## 三元条件运算符

Python提供了三元运算符，用来在某些简单双分支赋值情况。三元条件运算符语法格式如下：

条件为真的值 if 条件表达式 else 条件为假的值

案例：将上个案例用三元条件运算符表示：

```
num = input("请输入一个数字：")
print(num if int(num) < 10 else "数字太大")
```
